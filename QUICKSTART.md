# 🚀 מדריך התחלה מהיר - משאוושה v2.0

## ⚡ הרצה מהירה

```bash
npm install
npm run dev
```

האפליקציה תרוץ על: `http://localhost:5173/`

## הפעלה מהירה

### אפשרות 1: פתיחה ישירה בדפדפן
פשוט פתח את הקובץ:
```
public/index.html
```

### אפשרות 2: עם שרת מקומי (מומלץ)

**אם יש לך Node.js:**
```bash
npm start
```

**אם יש לך Python:**
```bash
cd public
python -m http.server 8000
```

ואז גש ל: `http://localhost:8000`

## תכונות חדשות בגרסה 2.0

✅ **שמירת טיוטות אוטומטית** - ההזמנה נשמרת אוטומטית  
✅ **ייצוא לאקסל** - ייצוא היסטוריה לקובץ CSV  
✅ **חיפוש מתקדם** - חיפוש בהיסטוריה ובמוצרים  
✅ **התראות Toast** - משוב ויזואלי משופר  
✅ **אנימציות חלקות** - חוויית משתמש משופרת  
✅ **ארכיטקטורה מודולרית** - קוד מאורגן וניתן לתחזוקה  

## מה השתנה מהגרסה הקודמת?

### ארכיטקטורה
- ✨ הקוד מפוצל ל-20+ קבצים מאורגנים
- ✨ קומפוננטים מודולריים וניתנים לשימוש חוזר
- ✨ הפרדה ברורה בין לוגיקה, נתונים ועיצוב

- ✨ התראות ומשוב ויזואלי
